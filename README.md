## TERMUX USER
```bash
> pkg update && pkg upgrade
> pkg install nodejs
> pkg install libwebp
> pkg install git
> pkg install ffmpeg
> pkg install imagemagick
> pkg install yarn
> termux-setup-storage
> git clone https://github.com/ArielSP1609/MyBot
> cd MyBot
> yarn
> npm start
```
